Accessibility Matters

Today I learned that not everyone uses the web the same way I do. Making websites accessible ensures that everyone can use them effectively. There are many ways to make a website more accessible, such as using proper semantic HTML, providing alternative text for images, and ensuring keyboard navigation works properly. For measuring accessibility, tools like WAVE and Lighthouse are very helpful.
